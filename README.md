Web scraping medical stackexchange

## Requirements
python 3.6

## Set-up 
`sh create_schema.sh`

`pip install -r requirments.txt`

`cp config.py.example config.py`

Fill in fields of config.py 

## Usage

`python main.py --num_pages {num_pages_to_scrape} [--update_tag_db]`
